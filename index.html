MOOD DIARY

나만의 감정 기록 도구로 마음 건강을 챙기세요

✨ 무료 버전 · made by 새벽숲

매일 5분, 내 마음을 돌보는 시간

MOOD DIARY는 당신의 감정과 기분을 쉽고 재미있게 기록할 수 있는 웹 도구입니다. 복잡한 설치 없이 브라우저에서 바로 사용하고, 모든 기록은 안전하게 보관됩니다.

📊 종합적인 기분 기록

하루의 감정을 다각도로 기록하세요

  • 오늘의 날씨와 기분 척도
  • 에너지 레벨 측정
  • 자기관리 체크리스트
  • 자유로운 감정 일기

📅 직관적인 달력 뷰

한 달간의 감정 변화를 한눈에 확인

  • 월별 기록 달력으로 보기
  • 날짜별 기분 이모지 표시
  • 클릭 한 번으로 상세 기록 확인
  • 감정 패턴 쉽게 파악

📈 월간 리포트 생성

내 감정 데이터를 분석한 맞춤 리포트

  • 월별 기분/에너지 평균 분석
  • 자기관리 실천 현황
  • 베스트 기분의 날 하이라이트
  • 이미지 또는 텍스트로 다운로드

🔒 안전한 개인정보 보호

내 기록은 오직 나만 볼 수 있어요

  • 모든 데이터 로컬 저장
  • 인터넷 없이도 사용 가능
  • 별도 회원가입 불필요
  • 개인정보 외부 전송 없음

🚀 지금 바로 시작하세요!

설치가 필요 없는 웹 버전으로 언제 어디서나 사용할 수 있어요

💾 MOOD DIARY 다운로드

📱 사용 환경

PC, 모바일, 태블릿 모든 기기에서 사용 가능
최신 브라우저 권장 (Chrome, Safari, Firefox)

💾 파일 정보

HTML 파일 1개 (약 50KB)
다운로드 후 브라우저로 열어서 사용

🔄 업데이트

새로운 기능이 추가되면 블로그에서 공지
기존 데이터는 그대로 유지

📖 사용법

1

다운로드

위의 다운로드 버튼을 클릭하여 HTML 파일을 받으세요

2

실행

다운로드한 파일을 브라우저로 열어주세요

3

기록

날씨, 기분, 에너지, 자기관리를 선택하고 일기를 써보세요

4

분석

달력에서 기록을 확인하고 월간 리포트를 받아보세요
