📊 MOOD DIARY

나만의 감정 기록 도구

매일의 기분, 날씨, 에너지 레벨을 기록하고
월간 리포트로 나만의 성장 과정을 확인해보세요

✨ 주요 기능

🌤️

날씨 & 기분 기록

날씨와 기분 척도(1-10)를 쉽게 기록하고 패턴을 발견하세요

⚡

에너지 레벨 추적

매일의 에너지 수준을 시각적으로 기록하고 관리하세요

🌱

자기관리 체크

운동, 명상, 수면 등 자기관리 활동을 체크하고 습관을 만들어가세요

📖

감정 일기

하루의 특별한 순간들을 자유롭게 기록해보세요

📅

달력 뷰

월별 기록을 한눈에 보고 나의 감정 패턴을 파악하세요

📊

월간 리포트

한 달간의 기록을 분석한 개인 맞춤 리포트를 다운로드하세요

💾 다운로드

🎯 무드 다이어리 완전판

모든 기능이 포함된 완전한 버전

📄 index.html 메인 페이지
🎨 css/style.css 스타일시트
⚙️ js/script.js 기능 스크립트

개별 파일 다운로드

🚀 사용 방법

1

파일 다운로드

위의 다운로드 버튼을 클릭하여 모든 파일을 받으세요

2

폴더 구조 유지

다운로드한 파일들을 같은 폴더에 놓고 구조를 유지하세요

📁 mood-diary/
├── 📄 index.html
├── 📁 css/
│ └── 🎨 style.css
└── 📁 js/
    └── ⚙️ script.js
3

index.html 실행

브라우저에서 index.html 파일을 열어서 사용하세요

🔒 개인정보 보호

모든 데이터는 사용자의 브라우저에만 저장되며, 외부로 전송되지 않습니다

📱 반응형 디자인

PC, 태블릿, 모바일 모든 기기에서 최적화된 환경을 제공합니다
